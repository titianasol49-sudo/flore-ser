PLANTAS MEDICINALES

¿Cuándo comprar?

Aloe Vera

Cicatrizante, hidratante. Para quemaduras o piel.

Tilo

Relajante, sedante natural. Té para el insomnio.

Romero

Estimulante, mejora la memoria. Infusiones o aromaterapia.

Menta

Digestiva, refrescante. Infusiones para el estómago o resfríos.

Manzanilla

Calmante, antiinflamatoria. Té para nervios o dolor de panza.

Boldo

Depurativa, digestiva. Té para el hígado.

Eucalipto

Descongestivo. Inhalaciones para resfríos.

Salvia

Antiséptica, digestiva. Infusiones medicinales.
